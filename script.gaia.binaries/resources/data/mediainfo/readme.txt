These are precompiled binaries of MediaInfo (https://mediaarea.net/en/MediaInfo).
The binaries are optional and Gaia will function normally, even if these are not installed.
They are used to detect additional metadata for local library, downloads, and cache files.

Current version: 20.09 (2020-11-11)

Linux (64bit):
	Tested!
	These are the binaries for Ubuntu 16.04. Hopefully they will working on other distros as well.
	The Ubuntu 16 binaries are used, since the newer versions require a newer GLIBC which is not available on older distros.
	- Executable: https://mediaarea.net/download/binary/mediainfo/20.09/mediainfo_20.09-1_amd64.xUbuntu_16.04.deb
	- Libmediainfo: https://mediaarea.net/download/binary/libmediainfo0/20.09/libmediainfo0v5_20.09-1_amd64.xUbuntu_16.04.deb
	- Libzen: https://mediaarea.net/download/binary/libzen0/0.4.38/libzen0v5_0.4.38-1_amd64.xUbuntu_16.04.deb
	- Libmms: https://packages.debian.org/buster/amd64/libmms0/download
